## css 样式命名统一一下

- 最外层采用当前文件名（组件名），命名时不分大小写，里层采用最外层的名字加-XXX 的形式，如果最外层名字太长就写前三个字母，在里层就又加个-，比如 home 组件外层我就叫 home，里面的我就叫 home-xxx
